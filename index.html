About

Overview

This tutorial introduces the theory and practice of decision-making for mobile robots using deep reinforcement learning, with emphasis on navigation, obstacle avoidance, policy generalization, and robotics-oriented evaluation.

The content is designed for researchers, graduate students, and engineers who want a structured path from MDP/POMDP foundations to deployable learning-based policies.

Who should attend?

  • Graduate students in robotics, AI, or control
  • Researchers working on navigation or embodied intelligence
  • Engineers building autonomous mobile robots
  • Practitioners interested in RL pipelines for real-world robotics

Prerequisites

Basic Python, machine learning fundamentals, and familiarity with robotics concepts such as localization, mapping, or motion planning.

Software setup

Python 3.10+, PyTorch, Gymnasium-compatible environments, and optional ROS 2 support for integration with mobile robot stacks.

Learning

Learning objectives

01

Understand the formulation of mobile robot decision-making as an RL problem.

02

Compare policy-gradient, actor-critic, and value-based approaches for navigation tasks.

03

Design reward functions, observation spaces, and evaluation metrics for robotic agents.

Content

Tutorial modules

Module 1

Foundations of RL for Robotics

MDPs, POMDPs, policies, value functions, exploration strategies, and robotics-specific formulation choices.

Module 2

State, Action, and Reward Design

Lidar/camera observations, low-dimensional states, discrete vs continuous actions, reward shaping, and sparse-reward considerations.

Module 3

Algorithms for Navigation

DQN variants, PPO, SAC, and actor-critic methods for reactive and goal-conditioned navigation.

Module 4

Simulation and Benchmarks

Environment design, curriculum learning, domain randomization, and evaluation protocols.

Module 5

Safety, Constraints, and Real Robots

Collision penalties, risk-sensitive policies, recovery behaviors, and runtime deployment considerations.

Module 6

Hands-on Repository Walkthrough

Code structure, training scripts, logging, reproducibility, and how to extend the baseline.
